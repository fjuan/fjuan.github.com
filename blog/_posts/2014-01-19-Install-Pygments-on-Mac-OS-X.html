How to install Pygments on OS X

From the Jekyll documentation page:
Jekyll also has built-in support for syntax highlighting of code snippets using Pygments
In order to install Pygments you only need next command: {% highlight bash %} sudo pip install Pygments {% endhighlight %} If you don't have pip command available: {% highlight bash %} sudo easy_install pip sudo pip install Pygments {% endhighlight %} You might find the next error message: {% highlight console %} Wheel installs require setuptools >= 0.8 for dist-info support. {% endhighlight %} That can be solved with: {% highlight bash %} sudo pip install setuptools --no-use-wheel --upgrade sudo pip install Pygments {% endhighlight %} Enjoy!! {% highlight ruby %} def show @widget = Widget(params[:id]) respond_to do |format| format.html # show.html.erb format.json { render json: @widget } end end {% endhighlight %}
